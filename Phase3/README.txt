Application 제작 환경
	Intellij IDE환경에서 postgreSQL driver를 통해 postgreSQL과 연동하여 진행


기능 설명 및 사용방법
최초 Application실행 시 1.로그인과 2.회원가입, 3.프로그램 종료를 선택한다.
1) 로그인 기능 시 Id와 Pw를 입력하며 해당 Id와 Pw정보를 DB에 조회하여 로그인 가능 여부를 판단하고 관리자인지 아닌지 권한을 확인 후 로그인토록 한다.
	1.1) 로그인 시 관리자 정보가 true이면 관리자 로그인으로 진행
		1.1.1) 관리자 로그인 시 [관리자 기능], [회원 기능] 으로 나뉨.

	1.2) 로그인 시 관리자 정보가 false이면 멤버 로그인으로 진행
		1.2.1) 멤버 로그인 시 회원 기능만 가짐.


[회원 기능]
	1. 회원 관련 기능 - 아래 중 원하는 기능의 숫자를 입력한다.
		1.1) 회원정보 수정 - 아래 중 수정하길 원하는 정보에 대한 숫자를 입력 후 수정 정보를 입력한다.
			1.1.1) user_name
			1.1.2) phone_num
			1.1.3) birth_date
			1.1.4) age
			1.1.5) gender
			1.1.6) address
			1.1.7) job
			1.1.8) mem_type


		1.2) 비밀번호 수정 - 수정하길 원하는 비밀번호를 입력한다.
		1.3) 회원 탈퇴 - 선택 시 회원 탈퇴가 이뤄진다.


	2. 영상물 관련 기능 - 아래 중 원하는 기능의 숫자를 입력한다.
		2.1) 영상물 전체 출력 - postgreSQL DB에 적재된 모든 영상물의 제목정보가 출력된다.
			2.1.1) 영상물 상세 정보 확인하기
				2.1.1.1) 상세 정보를 확인하려는 영상물의 번호 입력
			2.1.2) 뒤로가기 
				2.1.2.1) 평가하기
					2.1.2.1.1) 평가하기 : 평가할 영상물의 이름을 입력한 후, 점수를 입력한다.
					2.1.2.1.2) 거절하기
		2.2) 영상물 제목 검색 - 검색을 원하는 제목을 입력한다.
			2.2.1) 영상물 상세 정보 확인하기
				2.2.1.1) 상세 정보를 확인하려는 영상물의 번호 입력
			2.2.2) 뒤로가기 
				2.2.2.1) 평가하기
					2.2.2.1.1) 평가하기 : 평가할 영상물의 이름을 입력한 후, 점수를 입력한다.
					2.2.2.1.2) 거절하기

		2.3) 영상물 조건 검색 - 검색을 원하는 조건을 입력한다. 원하는 조건이 아닌 경우 enter로 넘어간다.
			2.3.1) 영상물 상세 정보 확인하기
				2.3.1.1) 상세 정보를 확인하려는 영상물의 번호 입력
			2.3.2) 뒤로가기 
				2.3.2.1) 평가하기
					2.3.2.1.1) 평가하기 : 평가할 영상물의 이름을 입력한 후, 점수를 입력한다.
					2.3.2.1.2) 거절하기

		*평가한 영상물의 정보는 검색 대상에서 제외된다.

	3. 평가 관련 기능 
		3.1) [회원 기능] 일 경우, 내가 평가한 영상물의 목록을 볼 수 있음.
		3.2) [관리자 기능] 일 경우, [평가된 모든 영상물의 목록] 또는 [본인이 평가한 영상물의 목록]을 볼 수 있음.


[관리자 기능]
	1. [관리자 기능] - 아래 중 원하는 기능의 숫자를 입력한다.
		1.1) 영상물 등록 - 영상물의 필수 등록 정보를 입력 후 추가정보는 원하는 기능의 숫자를 입력 후 해당 정보를 입력한다.
				* 필수정보 : title, type, runtime, start_year
			1.1.1) genre
			1.1.2) rating
			1.1.3) viewing_class
			1.1.4) account_id - id를 따로 입력할 필요 없이 현재 로그인 되어있는 관리자의 아이디가 자동으로 입력된다.

		1.2) 영상물 정보 수정 - 수정할 영상물의 제목을 입력한다.
			1.1) title
			1.2) type
			1.3) runtime
			1.4) start_year
			1.5) genre
			1.6) rating
			1.7) viewing_class
			
	2. 위의 [회원 기능]을 동작한다.


2) 회원가입 기능 시 관리자 계정 등록 여부 및 id, pw, 이름, 전화번호, 생년월일, 나이, 성별, 주소, 직업, 멤버 등급 등을 입력하고 DB에 새로운 ACCOUNT를 적재한다.
